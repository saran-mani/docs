# Synapsis Project Documentation

This repository contains a Docusaurus site used to track project documentation for the Synapsis (Spyk Health style) project.

## What is documented

- System and architecture overview
- Authentication flow
- Data flow between services/pages
- Page-level behavior and ownership
- Change log for docs updates

## Local development

```bash
yarn
yarn start
```

## Build

```bash
yarn build
```

## Documentation structure

- `docs/intro.md`: Main project summary
- `docs/architecture/*`: Technical architecture and interactions
- `docs/flows/*`: Auth and data flow documentation
- `docs/pages/*`: Page map and page-level behavior
- `docs/operations/change-log.md`: Documentation change tracking
